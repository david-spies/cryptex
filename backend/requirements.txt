# backend/requirements.txt
# =============================================================
#  CRYPTEX v2.2 — Python Backend Dependencies
#  Data source: CoinMarketCap Professional API
#
#  Install:
#    pip install -r requirements.txt
#
#  Or inside a virtualenv:
#    python3 -m venv venv && source venv/bin/activate
#    pip install -r requirements.txt
#
#  CMC API key setup:
#    1. Register free at https://coinmarketcap.com/api/
#    2. Copy your key into backend/.env as: CMC_API_KEY=your_key_here
#    3. Free "Basic" plan covers /listings/latest, /quotes/latest,
#       and /quotes/historical — sufficient for all dashboard features.
#    4. OHLCV historical (/ohlcv/historical) requires Standard plan+.
#       On Basic, the backend automatically falls back to constructing
#       approximate OHLC data from the quotes/historical endpoint.
# =============================================================
